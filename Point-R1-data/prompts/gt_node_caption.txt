你是一个 3D 部件标注员。现在需要为“当前节点部件”生成一个高质量中文 caption，必须能反映：
- 该部件的**可见外观/材质线索**（如金属、塑料、橡胶、玻璃、织物；光泽、透明、纹理等）
- **几何形状与结构**（轮廓、体积、曲面/棱边、孔洞/凹槽/凸起、对称性、长宽厚比例等）
- **空间位置与连接关系**（相对全局与父节点的位置；与子节点/邻接结构的连接方式）
- **功能推断**（基于形状/位置的合理推断，避免过度臆测）

你会看到多视角图像：每张图 **左侧是原始渲染图**，**右侧是用红色半透明高亮的当前节点区域**。

## 输入信息
- 全局物体：{global_name}
- 全局描述（可为空）：{global_caption}
- 父节点：{parent_name}
- 父节点描述（可为空）：{parent_caption}
- 当前节点名字（已确定，必须使用这个名字）：{current_name}
- 子节点名字列表（用于语义约束，可为空）：{child_names}

## 输出要求（严格 JSON）
只输出一个 JSON 对象，不要输出任何额外文本：
{
  "name": "{current_name}",
  "caption": "一段中文，2~5句，信息密集，避免空话",
  "visual_features": ["...","..."],
  "geometric_features": ["...","..."],
  "inferred_function": "一句话功能推断（可为空字符串）",
